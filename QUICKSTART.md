# ⚡ Quick Start (30 seconds)

## 🚀 Just Get It Running

### 1. Terminal 1: Backend
```bash
cd backend
npm start
```
Wait for: ✅ Real-time messaging initialized

### 2. Terminal 2: Frontend
```bash
cd frontend
npm run dev
```
Wait for: ✅ Compiled successfully

### 3. Browser: Open 2 Windows
- Window 1: http://localhost:5173
- Window 2: http://localhost:5173 (or 5174)

### 4. Login
- Login same or different users
- Chat starts working
